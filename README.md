# Resume Analyzer

A modern React + Tailwind CSS resume analyzer with:

- drag-and-drop PDF upload
- textarea-based resume input
- optional job description targeting
- `pdf.js` text extraction
- weighted keyword matching with synonyms
- section detection and experience-level detection
- role benchmarking across six predefined career paths
- advanced dashboard with score breakdowns, ATS risks, strengths, and roadmap suggestions
- animated score ring, matched/missing skills, and smart suggestions

## Tech

- React + Vite
- Tailwind CSS
- `pdfjs-dist`
- `lucide-react`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
